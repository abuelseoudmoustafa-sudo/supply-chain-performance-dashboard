# 🚚 Supply Chain Performance & Order Fulfillment Dashboard

## 📌 Project Overview

This Power BI dashboard analyzes supply chain performance and order fulfillment efficiency by monitoring procurement costs, supplier performance, delivery timelines, and order status.

The objective of this project is to help operations and procurement teams identify **delays, cost drivers, and supplier performance issues** to improve supply chain efficiency and decision-making.

## 📊 Key Metrics

- **Total Orders:** 150

- **Total Quantity Ordered:** 39K units  
- **Average Lead Time:** 23.99 days  
- **Delayed Orders:** 35.33%

---

## 🔍 Key Insights

### Supplier Cost Analysis
- **Supplier A** represents the highest procurement cost.
- Followed by **Supplier B** and **Supplier C**, indicating potential negotiation opportunities.

### Order Fulfillment Status
- **35.33% of orders are delayed**, indicating supply chain inefficiencies.
- On-time orders represent the majority but delays remain significant.

### Lead Time Trend
- Lead time fluctuates across months, dropping significantly in **March 2024** before stabilizing.
- This trend highlights potential operational improvements during that period.

### Product Procurement Analysis
- Certain products such as **Component X** and **Raw Material 1** account for a large share of supplier spending.
